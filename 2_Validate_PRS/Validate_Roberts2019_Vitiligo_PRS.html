plink_allele_frequencies <- geno_dat %>%
  mutate(across(everything(), as.numeric)) %>%
  select(starts_with("rs")) %>%
  pivot_longer(cols = everything(), names_to = "SNPtmp", values_to = "genotype") %>%
  group_by(SNPtmp) %>%
  summarize(sum=sum(genotype), len=length(genotype)*2) %>%
  ungroup() %>%
  mutate(
    SNP = str_split_fixed(SNPtmp, "_", 2)[, 1],  # Extract the SNP ID

    maf = sum / len

This plot tells us that there is reasonable-ish alignment between the MAFs estimated from our data and the MAFs from NCBI (mostly EUR-dominated cohorts, like TOPMED). We’re mainly only seeing potential allele mismatches at MAF around 0.5, which probably just reflects a little bit of variability in the MAF.

This is great, but doesn’t confirm that we’re scoring the correct allele since the risk allele is the weights allele and not necessarily the minor allele (why did we do that??).
